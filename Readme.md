<!-- Threejs Vite starter -->
 # (This is a starter pack for threejs with vite)

 ## Version
    - Threejs: 0.145.0
    - Vite: 3.1.0


## How to use
    - Clone the repo
    - Run `npm install`
    - Run `npm run dev`
    - Open `localhost:5137` in your browser

## How to build
    - Run `npm run build`
    - Open `localhost:${PORT}` in your browser
